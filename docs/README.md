# ВВЕДЕНИЕ

Настоящий отчет посвящен результатам прохождения проектной практики, в рамках которой велась разработка мобильного приложения для изучения русского языка как иностранного. Проект направлен на решение актуальной задачи интеграции иностранных студентов в русскоязычную среду через современные технологические решения.

# 1. Общая информация о проекте

Проект представляет собой разработку мобильного приложения **«Mishkova»**, предназначенного для обучения русскому языку. В основе проекта лежит использование современных программных инструментов для создания интерактивной образовательной среды.

## Название проекта

**Mishkova**

## Цели проекта

К 30 мая 2027 года создать функциональное и визуально привлекательное мобильное приложение для изучения русского языка как иностранного (РКИ). Целевая аудитория — студенты из стран Латинской Америки.

Основные задачи проекта:

1. Проектирование архитектуры приложения.
2. Разработка пользовательского интерфейса.
3. Внедрение механизмов автоматизированной проверки знаний.

## Задачи проекта

1. Создание дизайна и пользовательского интерфейса приложения.
2. Разработка комплектов заданий по типам:
   - чтение;
   - аудирование;
   - грамматика.
3. Проектирование и реализация серверной части (Backend):
   - разработка архитектуры приложения на языке Go;
   - создание структуры базы данных PostgreSQL для хранения прогресса пользователей и учебных материалов.
4. Проведение пользовательских исследований:
   - организация опросов;
   - тестирование прототипа среди студентов из Латинской Америки.
5. Разработка презентационного веб-сайта:
   - создание адаптивной посадочной страницы на HTML/CSS.
6. Ведение технической документации:
   - фиксация этапов разработки;
   - описание архитектурных решений в формате Markdown;
   - использование системы контроля версий Git.
7. Обеспечение кроссплатформенности и адаптивности интерфейса.
8. Тестирование и отладка системы:
   - функциональное тестирование;
   - нагрузочное тестирование.

# 3. Описание задания по проектной практике

Задание на практику включало два основных блока.

## Обязательная часть

- Создание презентационного веб-сайта на базе HTML/CSS (не менее 5 страниц), включающего разделы:
  - «Аннотация»;
  - «О проекте»;
  - «Участники»;
  - «Журнал прогресса»;
  - «Ресурсы».
- Ведение проектной документации с использованием синтаксиса Markdown.
- Размещение исходного кода и документации в системе контроля версий Git (GitHub).

## Вариативная часть

Разработка большой языковой модели (LLM) с нуля на языке Python с использованием фреймворка PyTorch.

Результатом данной разработки является создание архитектуры модели GPT-2, подготовленной для дальнейшего развития и обучения.

Проведенные опросы среди студентов из Латинской Америки позволили адаптировать контент модели под культурные и языковые особенности испаноязычной аудитории.
